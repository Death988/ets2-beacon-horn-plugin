## 📋 What is ets2-beacon-horn-plugin?

ets2-beacon-horn-plugin lets you control the beacon light and horn in Euro Truck Simulator 2 using a simple key press. Hold the **J** key to turn the beacon light on or off, and sound the horn at the same time. This makes it easier to signal while driving without using the game’s menu or multiple buttons.

This software works through the telemetry plugin system in ETS2. It reads your game state and sends commands to activate the beacon and horn.

---

## 🔍 Features

- Use the **J** key to toggle beacon lights and horn together.
- Works with both Euro Truck Simulator 2 and TruckersMP multiplayer.
- Simple and lightweight plugin with low system impact.
- Supports the official telemetry plugin system used by SCS Software.
- No need to configure multiple buttons in the game.
- Easy to install and run on Windows.

## 🛠 How to Use the Plugin

Once the plugin is installed and ETS2 is running with telemetry enabled:

- Hold **J** to switch your beacon light on or off.
- The horn will sound at the same time you turn the beacon on or off.
- Pressing **J** again will turn the beacon off and stop the horn.

You do not need to open any menus or use any other keys. This allows you to keep your hands on the wheel while signaling.

## ⚙ Troubleshooting

If the plugin does not work as expected:

The plugin comes with a simple configuration file (`config.json`) if you want to change which key toggles the beacon and horn.

Example:

```json
{
  "toggleKey": "J"
}
```

Edit this file with any text editor to choose a different key if you prefer.
