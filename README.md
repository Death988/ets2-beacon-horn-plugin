# ETS2 Beacon & Horn Plugin

Telemetry plugin for Euro Truck Simulator 2 that lets you control the beacon and horn using the **J** key.

When you hold **J**:
- Beacon toggles on press and off on release  
- Horn sounds while the key is held  

---

## 📦 Installation

1. Download `BeaconPluginByPolyakoff.dll`
2. Move it to: `Program Files (x86)\Steam\steamapps\common\Euro Truck Simulator 2\bin\win_x64\plugins`
3. Start the game
